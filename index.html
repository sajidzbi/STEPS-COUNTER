About This App

SM Steps Counter Pro v3.0 — A comprehensive fitness tracking app with real-time motion sensor integration, personalized calorie calculation, and detailed analytics.

Features include accelerometer-based step detection, MET-formula calorie burn, weekly/monthly charts, achievement badges, water tracking, BMI calculator, and more.

Built with HTML5 · Chart.js · DeviceMotion API · localStorage persistence
